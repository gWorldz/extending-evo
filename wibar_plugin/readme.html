Wibiya is a web toolbar and application platform that you can easily install on your site. Add a large variety of apps that will boost your site’s functionality and reach.

The Wibiya toolbar is interactive and user friendly. Creating an account is easy and free, you can be well on your way to installing the wibiya toolbar on your site within minutes.

No technical know-how necessary. The Wibiya solution makes it possible to add exciting features to your site without any code or programming knowledge.

At the bottom of this page is a fully functional Wibiya Toolbar that can be used for plugin support and to give you an idea of what is possible. The Wibiya Toolbar install code looks something like this:

<script src="http://cdn.wibiya.com/Toolbars/dir_0255/Toolbar_255733/Loader_255733.js" type="text/javascript"></script>
<noscript><a href="http://www.wibiya.com/">Web Toolbar by Wibiya</a></noscript>

Rather than using the entire install code, the plugin only requires either the script src or your 6 digit user ID number (255733 in the example above).

Instructions:

  1. Upload the plugin folder to your ../plugins/ folder
  2. Install the plugin
